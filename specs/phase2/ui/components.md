# UI Components
 
## General
- Follow Material Design principles
- Responsive design for various screen sizes
 
## Components
 
### Task Card
- Displays task title, status, and due date
- Allows editing and deleting tasks
 
### Task List
- Displays a sortable and filterable list of Task Cards
- Shows a "Create New Task" button
 
### Navigation Bar
- Contains branding and user authentication status
- Links to different sections of the application
 
### Forms
- Standardized input fields and buttons
- Client-side validation
