# drizzle-orm

## Description
Lightweight TypeScript ORM for PostgreSQL with zero-dependency runtime and SQL-like API.

## Core Principles
1. **Type Safety**: Full TypeScript inference without runtime overhead.
2. **SQL-Like API**: Write queries that look like SQL.
3. **Lightweight**: Minimal bundle size, no heavy dependencies.
4. **Migrations**: First-class migration support with CLI tooling.
